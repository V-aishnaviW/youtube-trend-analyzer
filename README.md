# YouTube Trend Analysis Dashboard

## Project Overview
This project analyzes YouTube trending video data using SQL and Power BI to identify audience engagement patterns, content performance, hashtag effectiveness, publishing trends, and channel growth behavior.

The goal of this project is to help content creators and businesses understand what type of content performs best and how audience engagement can be improved using data-driven insights.

---

## Tools & Technologies Used
- SQL
- Power BI
- DAX
- Data Modeling
- ERD Design

---

## Business Problem
Content creators often struggle to identify:
- Which video categories perform best
- What type of content drives engagement
- Which hashtags improve visibility
- Which upload patterns support growth
- What characteristics are common among viral videos

This project aims to solve these problems through data analysis and dashboard storytelling.

---

## Dataset Information
The dataset contains:
- Videos
- Channels
- Categories
- Hashtags
- Video performance snapshots

The project includes over 1000+ records across multiple relational tables.

## Key Insights
- Entertainment and music content generated the highest audience engagement.
- Comedy and nostalgic short-form videos showed strong growth trends.
- Consistent upload frequency improved channel performance.
- Some videos achieved high views but lower engagement, indicating passive audience behavior.
- Videos using multiple hashtags received better visibility.
- Tuesday showed the highest average views in the dataset, though influenced by lower upload volume.

## Challenges Faced
During the project, several challenges were encountered:
- Ambiguous relationships in Power BI
- Incorrect hashtag filtering behavior
- Day-wise sorting issues
- Data model debugging and filter propagation problems

These issues were resolved through proper relationship management and DAX optimization.
